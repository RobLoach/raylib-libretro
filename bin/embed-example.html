raylib-libretro embed example

This page exercises every usage form of embed.js. Serve it from the same directory as the built index.js / index.wasm / index.data (and cores/), e.g. python3 -m http.server in the install dir. Drop a ROM named game.nes next to this file, or pass ?game=<url>&core=<name> in this page's URL.

1. Marker element

<div data-raylib-libretro data-game="game.nes"></div>
<script src="embed.js"></script>

2. Programmatic API

RaylibLibretro.embed('#game2', { game: 'game.nes', core: 'fceumm' });

3. Self-configuring single tag

The most minimal form — one tag, placed where you want the game. (Shown as code here so it doesn't add a third instance; only one emulator can run per page.) 

<script src="embed.js" data-game="https://example.com/mario.nes"></script>
